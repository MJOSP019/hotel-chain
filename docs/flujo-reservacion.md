# Flujo principal de reservación

1. El usuario ingresa al portal web.
2. Realiza una búsqueda por ciudad, fechas y número de huéspedes.
3. El sistema filtra los hoteles y habitaciones disponibles.
4. El usuario selecciona una opción.
5. Se genera una reservación.
6. El usuario completa el proceso de checkout.
7. El sistema confirma la reservación.
8. Se genera un PDF de confirmación.
9. El usuario puede consultar su historial de reservaciones.
