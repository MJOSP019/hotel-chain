# Sistema Cadena de Hoteles

## Descripción general
Sistema web para la gestión de una cadena de hoteles, que permite búsqueda de habitaciones, reservaciones, checkout, administración e integración REST.

## Tecnologías utilizadas
- .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- ASP.NET Core Identity
- JWT Authentication
- Blazor WebAssembly

## Herramientas de documentación técnica
- Swagger / OpenAPI para documentación de endpoints REST
- DocFX para documentación técnica del proyecto
