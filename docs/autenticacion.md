# Autenticación y autorización

## Tecnologías utilizadas
- ASP.NET Core Identity
- JWT Authentication

## Roles principales
- ADMIN
- REGISTERED
- WEBSERVICE

## Descripción
El sistema controla el acceso a funcionalidades según el rol del usuario.  
Los endpoints administrativos requieren autenticación y autorización basada en roles.  
La integración REST utiliza usuarios de tipo WebService autenticados mediante JWT.
