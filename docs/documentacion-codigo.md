# Documentación técnica generada del código

## Objetivo
Como parte de la preentrega, se generó documentación técnica del código utilizando herramientas acordes al stack del proyecto.

## Herramientas utilizadas
- **Swagger/OpenAPI** para documentación automática de endpoints REST del backend.
- **DocFX** para publicar la documentación técnica del proyecto.
- **Comentarios XML de C#** para generar archivos `.xml` de documentación durante la compilación.

## Backend documentado
Se documentaron elementos reales del código backend, incluyendo:
- AuthController
- ReservationsController
- AdminHotelsController
- HotelDetailDto
- Hotel
- Reservation

## Frontend documentado
Se documentaron componentes y lógica del frontend, incluyendo:
- Search.razor
- Checkout.razor

## Resultado
Durante la compilación del proyecto se generaron archivos XML de documentación, lo cual permite evidenciar que la documentación no fue únicamente escrita manualmente, sino también generada a partir del código fuente documentado.
