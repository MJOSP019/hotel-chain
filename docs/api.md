# API REST

## Descripción
La API REST expone los servicios funcionales del sistema y está documentada mediante Swagger/OpenAPI.

## Principales grupos de endpoints
- Auth
- AdminHotels
- AdminRooms
- AdminRoomInventories
- AdminReservations
- AdminUsers
- AdminAnalytics
- Cities
- HotelReviews
- Reservaciones públicas
- Integración REST

## Uso
Los endpoints permiten realizar operaciones de consulta, administración y reservación dentro del sistema de cadena de hoteles.
