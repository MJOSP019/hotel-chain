# Frontend

## Descripción
El frontend fue desarrollado con Blazor WebAssembly y permite la interacción de usuarios visitantes, registrados y administradores con el sistema.

## Páginas principales
- Home
- Login
- Register
- Search
- HotelDetail
- Checkout
- MyReservations
- AdminHotels
- AdminRooms
- AdminInventories
- AdminReservations
- AdminUsers
- AdminAnalytics

## Documentación técnica generada del código
En el frontend se documentaron componentes y lógica interna mediante comentarios dentro del código Razor y estructuras auxiliares del proyecto.

Además, la documentación técnica del frontend se integró en DocFX para describir:
- rutas y páginas principales,
- flujo de búsqueda,
- flujo de checkout,
- interacción con la API REST,
- uso de JWT para autenticación.
