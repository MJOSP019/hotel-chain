# Módulos principales del sistema

## Módulo de usuarios
Permite registro, login, asignación de roles y autenticación con JWT.

## Módulo de hoteles
Permite administrar hoteles y su información general.

## Módulo de habitaciones
Permite administrar habitaciones asociadas a cada hotel.

## Módulo de inventario
Permite controlar disponibilidad de habitaciones por fecha.

## Módulo de reservaciones
Permite crear, confirmar, consultar, cancelar y modificar reservaciones.

## Módulo de analíticas
Registra búsquedas realizadas desde portal web e integración REST.

## Módulo de integración REST
Permite a clientes empresariales consultar hoteles, disponibilidad y reservaciones mediante endpoints protegidos.
